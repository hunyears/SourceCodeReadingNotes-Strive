一：架构分层

    A[你的代码] -->|调用| B(Feign接口)
    B -->|动态代理| C[MethodHandler]
    C -->|编码| D[RequestTemplate]
    D -->|发送| E[HTTP Client]
    E -->|解码| F[返回结果]
    
二：关键组件

    Feign.Builder: 核心构建器（配置编解码器、重试等）
    Contract: 注解解析（支持Spring MVC注解）
    Encoder/Decoder: 请求/响应编解码（如JSON转换）
    Client: HTTP客户端实现（默认JDK，可替换为OkHttp等）
    Retryer: 重试策略（指数退避等）
    RequestInterceptor: 请求拦截器（添加认证头等）
    
三：执行流程

    1. 动态代理拦截方法调用 → 
    2. 通过MethodHandler处理 → 
    3. 构建RequestTemplate → 
    4. 应用拦截器 → 
    5. 通过Client发送请求 → 
    6. 解码响应或处理异常
